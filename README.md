## Project Description
This is an Android Studio project built in Java. It demonstrates the use of multiple Android UI components together with SQLite database storage and a second activity opened with an Intent.

### Main Features
- EditText fields for user input
- Button actions for submit, delete, and display
- Spinner for selection
- AutoCompleteTextView for searchable input
- RadioButtons and CheckBoxes for choices
- SQLite database for saving and deleting data
- Multiple views using `MainActivity` and `MainActivity2`
- Data display on screen

## How to Run the Project
1. Open Android Studio.
2. Click **Open** and select the project folder `Databasewithcomponents`.
3. Wait for Gradle to sync and finish indexing.
4. Connect an Android device or start an emulator from **Device Manager**.
5. Select the device from the top device menu.
6. Click **Run** ▶️.
7. Enter the form data in the app and use the buttons to test the functionality.

## Notes
- The project is written in Java.
- It uses SQLite as the local database.
- The second screen is opened using an Intent from the main screen.
